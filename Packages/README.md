# 🏓 Pong Game

A simple Pong game built with Unity.

## 🎮 Features
- Two-player gameplay
- Score tracking
- Winner screen
- Restart game

## 🛠️ Built With
- Unity
- C#

## 🚀 How to Play
- Player 1:
  - W = Up
  - S = Down
- Player 2:
  - Up Arrow = Up
  - Down Arrow = Down

The first player to reach the target score wins.
